Word Quiz Help

  • Type the correct answer and press Enter (or click "Submit")
  • If it's correct, you'll move to the next word. (For Conjugation, you need to get all six answers right)
  • Type "a" if you want to be marked correctly anyway.
  • Type a number (e.g. 5) and press Enter to jump to that word in the list.
  • Use diacritic buttons (é, è, etc.) if needed.
  • In test mode, you will be quizzed on every word from the chapter in random order. If you get everything right in a cycle test mode will automatically terminate.
  • In test mode, missed answers are tracked and will be tested twice in the next cycle.
  • You can toggle between word/definition with the Swap button.
  • If you toggle the Random button, the next word will be a random word from the chapter.
